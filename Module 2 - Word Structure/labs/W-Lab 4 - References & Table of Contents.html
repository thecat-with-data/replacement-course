W-Lab 4: References & Table of Contents

Purpose

This lab focuses on managing references, inserting citations, and generating automated reference tables such as a Table of Contents and Table of Figures.

Outcomes

  • A11 Create and manage references
  • A12 Create and manage reference tables

Instructions

  1. Download the W-Lab 4 starter document.
  2. Follow all task prompts using Word's References tools.
  3. Ensure all automated tables are updated before submission.
  4. Upload your completed document.

Required Tasks

  • Insert citations using Word's citation manager.
  • Generate a bibliography or works cited page.
  • Insert a Table of Contents.
  • Add captions to tables or figures.
  • Create a Table of Figures.
