Part 1: What is a Named Range?

A named range is a label assigned to:

  • A single cell
  • A group of cells (range)

Once named, the range can be referenced by name instead of by cell address.

Examples

  • Sales_2026
  • Expenses_2026
  • Monthly_Revenue
  • Tax_Rate

Names should be:

  • Descriptive and meaningful
  • Unique within the workbook
  • Easy to remember
  • Free of spaces (use underscores instead)
