Lesson 1: Manipulating Data in Worksheets

By the end of this lesson, you will be able to:

  • Edit and arrange data accurately within worksheets
  • Insert, delete, and move rows and columns safely
  • Sort and filter data correctly
  • Identify common data-manipulation mistakes
  • Preserve data integrity while making structural changes

Why Data Manipulation Matters

Once data is imported, formatted, and view-ready, the real work begins.

In professional Excel usage, users must routinely:

  • Add new records
  • Remove outdated entries
  • Rearrange columns
  • Reorder rows
  • Prepare data for analysis

Small manipulation errors can corrupt datasets and invalidate analysis.

Key mindset shift:

In Excel, how you manipulate data is just as important as what the data says.
