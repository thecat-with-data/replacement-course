Part 4: Borders, Shading, & Visual Grouping

Borders and shading can be powerful, when used sparingly.

Appropriate Uses

  • Outline a data table
  • Emphasize header rows
  • Separate sections conceptually

Avoid

  • Heavy borders everywhere
  • Multiple colors without meaning
  • Decorative shading that doesn't communicate structure

Visual grouping should support understanding, not overwhelm.
