Part 6: Structured References (Core Concept)

Tables introduce a new way to reference data:

Cell Reference (Old Way)

=SUM(B2:B100)

Structured Reference (Table Way)

=SUM(Sales[Amount])

Benefits:

  • Easier to read
  • Self-explanatory
  • Automatically adjusts as data grows
  • Reduces referencing errors

Structured references connect directly to column names instead of cell positions.
