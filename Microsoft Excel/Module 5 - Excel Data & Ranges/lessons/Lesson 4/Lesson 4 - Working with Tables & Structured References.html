Lesson 4: Working with Tables & Structured References

By the end of this lesson, students will be able to:

  • Convert a data range into an Excel table
  • Apply and modify table styles
  • Sort and filter data using table tools
  • Understand how tables expand automatically
  • Use structured references to refer to table data
  • Explain why tables are preferred over raw data ranges

Why Tables Matter in Excel

Up to this point, you've worked with ranges of cells.

Now you move to Excel Tables, which are powerful and easier to manage.

Tables:

  • Maintain consistent structure
  • Automatically expand as data grows
  • Support filtering and sorting
  • Work seamlessly with formulas and charts

Key concept:

A table turns raw data into a managed data system.
