Lab 2 - Formatting Worksheets & Customizing Views

Purpose

This lab develops your ability to format Excel worksheets professionally and customize views for readability, printing, and collaboration. You will apply appropriate number formats, adjust layout and alignment, freeze panes, and prepare a workbook that is easy for others to use.

These skills are foundational for all future Excel work involving formulas, tables, and analysis.

Learning Outcomes

Upon completion of this lab, you will demonstrate the ability to:

  • Format worksheets and workbooks appropriately (B21)
  • Customize Excel options and views (B22)
  • Prepare content for collaboration and sharing (B23)

Required Tasks

Complete all parts of this lab. Each part is graded.

Number Formatting

Apply appropriate number formats to the data:

  1. Identify at least three different data types in the worksheet, such as:
    • Numbers
    • Currency
    • Dates
    • Percentages
    • IDs or codes
  2. Apply the correct Number Format to each column.
  3. Ensure that:
    • Monetary values display consistently
    • Dates are readable and uniform
    • ID columns retain leading zeros (if applicable)

Formatting must match the data's meaning—not just its appearance.

Layout, Alignment & Readability

Improve worksheet readability by adjusting layout:

  1. AutoFit column widths so all values are visible.
  2. Adjust row heights only if necessary.
  3. Apply cell alignment appropriately:
    • Text left-aligned
    • Numbers right-aligned
    • Headers aligned clearly
  4. Bold the header row to distinguish it from data.
  5. Ensure there are no blank rows inside the dataset.

Worksheet Organization

  1. Confirm the worksheet has a clear and meaningful name.
  2. If a second worksheet exists (e.g., Notes):
    • Ensure it is named clearly
    • Confirm it does not interrupt the data grid
  3. Remove unnecessary blank worksheets if present.

Professional organization improves collaboration and reduces errors.

Customizing Views

Configure the worksheet for ease of use:

  1. Freeze panes so:
    • The header row remains visible when scrolling
  2. Switch between:
    • Normal View
    • Page Layout View
    • Page Break Preview
  3. Return to Normal View when finished.

Demonstrate that you can intentionally change views based on task needs.

Print Preparation

  1. Prepare the worksheet for printing:
    • Open Print Preview.
  2. Adjust:
    • Orientation (portrait or landscape)
    • Scaling so the data fits appropriately
  3. Ensure:
    • Header row repeats on printed pages (if applicable)
    • Content is readable on paper

You do not need to print—only prepare and preview.

Collaboration Readiness

Before submission:

  1. Review worksheet for:
    • Consistent formatting
    • Clear headers
    • Logical layout
  2. Save the workbook using the required file name.
  3. Confirm the workbook would be understandable to another user.

Grading Checklist

Formatting

  • Appropriate number formats applied (B21)
  • Headers clearly formatted
  • Consistent alignment and spacing

Views & Layout

  • Freeze panes used correctly (B22)
  • Worksheet views demonstrated
  • Print layout reviewed and adjusted

Organization & Collaboration

  • Worksheets named clearly
  • Layout is readable and professional (B23)
  • Correct file naming used

Common Issues to Avoid

  • Leaving columns formatted as "General"
  • Mixing number formats within a column
  • Forgetting to freeze headers
  • Printing without previewing
  • Overusing color or cell merging
  • Leaving default worksheet names

Reflection

Before submitting, ask yourself:

  • Can someone else understand this worksheet instantly?
  • Would this be usable in a meeting or report?
  • Does formatting support clarity rather than distract from it?
