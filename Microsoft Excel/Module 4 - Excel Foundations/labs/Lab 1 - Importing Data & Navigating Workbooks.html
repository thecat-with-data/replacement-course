E-Lab 1: Importing Data & Navigating Workbooks

Purpose

This lab introduces you to working with data in Excel. You will practice importing data from an external file, navigating worksheets and workbooks efficiently, and reviewing imported data for accuracy.

This lab builds the foundational skills required for all future Excel work in this course.

Learning Outcomes

Upon completion of this lab, you will demonstrate the ability to:

  • Import data into an Excel workbook (B19)
  • Navigate effectively within worksheets and workbooks (B20)
  • Recognize common data-import issues
  • Organize and save workbooks using professional conventions

Getting Started

  1. Download the E-Lab 1 data file provided by your instructor (CSV or TXT).
  2. Open Microsoft Excel.
  3. Import the file into a new workbook.
  4. Save the workbook immediately as:
    • Lastname_Firstname_E-Lab1.xlsx

Required Tasks

Complete all parts of this lab. Each part is graded.

Importing Data

  1. Import the provided data file into Excel using the appropriate import method.
  2. Preview the data before completing the import.
  3. Confirm that:
    • Data appears in multiple columns, not a single column.
    • Each column contains a single type of data.
    • No rows or columns are missing.

If data imports incorrectly, re-import it rather than trying to "fix" it manually.

Reviewing Imported Data

After import, review the worksheet carefully:

  • Identify the header row.
  • Verify that rows represent individual records.
  • Confirm that values appear reasonable and aligned correctly.
  • Look for common problems, such as:
    • Dates shown as text
    • Missing values
    • Extra blank rows or columns

DO NOT correct issues yet—just confirm their presence or absence.

Navigating within the Worksheet

Demonstrate effective navigation skills:

  1. Use keyboard shortcuts (arrow keys, Ctrl+Arrow) to move through the data.
  2. Use Ctrl + Home and Ctrl + End to identify data boundaries.
  3. Use the Name Box to jump to a specific cell (e.g. A1, D50).
  4. Use Ctrl + G / Go To to jump to a specific location.

These actions prepare you for working with large datasets.

Working with Worksheets

  1. Rename the imported worksheet to a meaningful name related to the data.
  2. Insert a second worksheet.
  3. Rename the second worksheet (e.g. Notes or Observations).
  4. Navigate between worksheets using worksheet tabs.

Proper worksheet management is essential for multistep Excel work.

Selection Accuracy

Demonstrate correct selection techniques:

  1. Select:
    • A single cell
    • A contiguous range
    • A full column
  2. Select two non-adjacent ranges using the Ctrl key.

Selection accuracy affects all formatting, sorting, and analysis in Excel.

Grading Checklist

Data Import

  • Data imported successfully (B19)
  • Columns separated correctly
  • No missing or collapsed data

Navigation

  • Worksheet renamed logically (B20)
  • Navigation tools used correctly
  • Second worksheet added and renamed

Organization & Accuracy

  • Data structure reviewed
  • Selections performed correctly
  • Professional file naming used

Common Issues to Avoid

  • Importing data into a single column
  • Leaving worksheet names as "Sheet1"
  • Ignoring incorrect imports
  • Selecting partial or incorrect ranges
  • Rushing to format or analyze data

Reflection

Before submitting, consider:

  • Could someone else understand this workbook at a glance?
  • Could you navigate this file easily one week from now?
  • Is the data clearly organized and intact?
