Part 6 - The Name Box and Go To Tool

Professional Excel users utilize the Name Box and Go To Tool for maximum efficiency.

Name Box

  • Displays the address of the current cell
  • Can be used to jump to a specific cell or range.

Example:

  • Type D250 then press Enter. Notice what happens.
  • Use the keyboard shortcut to take you back to the first cell of the worksheet.

Go To Tool

Accessed via:

  • Ctrl + G or F5

Allows jumping to:

  • Specific cell reference
  • Named ranges (covered in a later lesson)
  • Other objects

These tools are especially useful for large datasets.
