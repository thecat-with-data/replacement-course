Part 5 - Column Width, Row Height & Alignment

Column Width

  • AutoFit columns to reveal full values
  • Avoid overly wide or narrow columns

Row Height

  • Adjust manually only when needed
  • Avoid oversized rows unless intentional

Alignment

  • Text: usually left-aligned
  • Numbers: usually right-aligned
  • Headers: centered or left-aligned

Proper alignment helps readers scan data accurately.
