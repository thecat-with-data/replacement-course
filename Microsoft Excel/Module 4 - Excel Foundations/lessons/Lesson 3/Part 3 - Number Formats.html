Part 3 - Number Formats

Number formats help viewers correctly interpret data.

Common Number Formats

  • General—default (often unclear)
  • Number—numeric values with decimals
  • Currency—monetary values
  • Accounting—aligned currency for reports
  • Percentage—rates and proportions
  • Date—calendar values
  • Text—values Excel should not calculate

Best Practices

  • Always format before sharing data
  • Use currency and date formats consistently
  • Apply text format to IDs or codes with leading zeros
