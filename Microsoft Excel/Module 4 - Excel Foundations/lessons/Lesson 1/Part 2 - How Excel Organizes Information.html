Lesson 1 Part 2: How Excel Organizes Information

Before importing data, it's essential to understand Excel's structure.

Workbook

  • A single Excel file (.xlsx)
  • Can contain multiple worksheets

Worksheet

  • A grid of rows and columns
  • Used to store and manipulate data

Cells

  • Intersection of a row and column
  • Each cell stores:
    • A value (number, text, date, etc.)
    • Or a formula (later modules)

Excel is a structured grid, not a text document. Every piece of data must live in a cell.

Rows, Columns & Data Direction

Professional Excel data follows patterns:

  • Rows represent individual records (e.g. one student, one sale)
  • Columns represent fields or attributes (e.g. name, date, amount)

Consistent structure allows Excel to:

  • Sort data
  • Filter information
  • Calculate accurately

If structure is inconsistent, Excel's tools may fail or produce incorrect results.
