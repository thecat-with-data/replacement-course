Lesson 1: Importing Data & Understanding Excel's Structure

By the end of this lesson, you will be able to:

  • Explain how data is organized in Excel workbooks and worksheets
  • Import data from an external source (such as a CSV file)
  • Identify common data-import issues and correct them
  • Distinguish between data values and data appearance
  • Prepare imported data for formatting and analysis

Why Data Import Matters

In the real world, data rarely starts inside Excel.

Most Excel users work with data that comes from:

  • Surveys
  • Accounting systems
  • Databases
  • Websites
  • Other software applications

If data is imported incorrectly, every calculation, table, and chart built on it will be wrong.

This lesson focuses on the idea that Excel analysis begins at import—not at formulas.
