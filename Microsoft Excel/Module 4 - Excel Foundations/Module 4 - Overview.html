Module 4: Excel Foundations

Module Purpose

This module introduces you to Microsoft Excel as a data-driven application. While Word focused on document creation and presentation, Excel shifts the emphasis to working with data, understanding how information is stored, manipulated, and prepared for collaboration and analysis. This module establishes the core habits and concepts students need before moving on to formulas, calculations, tables, and charts in later Excel modules.

By the end of this module, you will be able to:

  • Import data from external sources into Excel workbooks
  • Navigate efficiently within worksheets and workbooks
  • Format worksheets and workbooks for readability and usability
  • Customize Excel views and options
  • Configure workbooks for collaboration and sharing

This module supports Excel learning outcomes B19-B23

Conceptual Shift: Word to Excel

Consult the following table to transition from Word to Excel:

Word Excel
Content flows top-to-bottom Data lives in rows and columns
Formatting communicates ideas Structure enables calculation
Errors are visible while reading Errors may hide until used
Documents are mostly static Workbooks are dynamic

Module 4 is about learning how Excel thinks before asking Excel to solve problems.
