Module 1: Word Foundations

Welcome to Module 1: Word Foundations! In this module, we will explore the fundamentals of Microsoft Word.

By the end of this module, you should be able to:

  • Navigate a Word document using built-in tools.
  • Format text and paragraphs using standard conventions.
  • Insert new text, lists, and page breaks.
  • Save, version, export, and share Word documents.
  • Run Document Inspector and Accessibility Checker to prepare documents for submission.
