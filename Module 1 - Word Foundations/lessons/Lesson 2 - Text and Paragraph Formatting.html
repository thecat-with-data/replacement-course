Lesson 2: Text & Paragraph Formatting Essentials

This lesson teaches foundational text and paragraph formatting.

1. Text Formatting

  • Bold, italicize, underline
  • Change font size and color
  • Use highlighting sparingly

2. Paragraph Formatting

  • Alignment: left, center, right, justify
  • Spacing: set line spacing to 1.15
  • Indentation: first-line or hanging indents

3. Lists

  • Create bulleted, numbered, and multilevel lists
  • Practice nesting list items

4. Practice

Reformat several paragraphs using these tools to create a clean layout.
