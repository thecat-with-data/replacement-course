Lesson 2: Text & Paragraph Formatting Essentials

This lesson teaches foundational text and paragraph formatting.

By the end of this lesson, you will be able to:

  • Apply common text formatting conventions in professional documents.
  • Modify paragraph alignment, spacing, and indentation.
  • Create and format bulleted, numbered, and multilevel lists.
  • Understand when and why different formatting choices are appropriate.
  • Use formatting tools instead of manual, error-prone techniques.

What Formatting Actually Means in Word

Formatting in Microsoft Word controls how your content looks and behaves. It goes beyond just "making things pretty".

Proper formatting affects:

  • Readability
  • Accessibility
  • Consistency
  • Professional appearance
  • Automatic tools (Navigation Pane, Table of Contents, etc.)

Formatting Tools and Best Practices

Tool Purpose Avoid When
Bold Emphasis, section labels, keywords Entire paragraphs
Italic Titles, emphasis, technical terms Replacing proper headings
Underline URLs, forms Body text (poor accessibility)
Font Size Headings, emphasis Inconsistent "random" resizing
Font Color Highlights in drafts Color-only meaning (bad for accessibility)

Paragraph Formatting

Paragraph formatting affects entire blocks of text at once.

  1. Alignment

    • Left (recommended) - default for most documents
    • Center - titles, section breaks
    • Right - rarely useful outside page numbers
    • Justify - optional for long reports; can cause odd spacing

  2. Line Spacing

    3. For academic/professional documents:

    • 1.15 is the modern standard.
    • Double spacing remains common in academic writing.

  3. Paragraph Spacing

    4. Use:

    • Spacing before: 6-12 pt
    • Spacing after: 6-12 pt

    Always avoid: Adding blank lines to create spacing (hard to maintain, breaks flow).

  4. Indentation

    5. Use for structured text:

    • First Line Indent - for body paragraphs
    • Hanging Indent - for references/bibliographies

    These tools ensure consistent formatting across the full document.

Lists (Bulleted, Numbered, & Multilevel)

Use lists to organize information in a structured way.

  • Bulleted Lists

    • Use for:

    • Ideas
    • Examples
    • Non-sequential items
  • Numbered Lists

    • Use for:

    1. Steps
    2. Procedures
    3. Instructions
  • Multilevel Lists

    • Use when you need to show hierarchy or nested information.

    • A main idea
      • With sub-points
        • And sub-sub points.

Try it!

  2. Highlight the second item
  3. Click Increase Indent to create a sub-point.
  4. Change the bullet style if needed
  5. You have now created a multilevel list.

Common Mistakes Beginners Make

  • Using spaces or tabs to align text

    • Use the formatting tools instead.

  • Using multiple Returns (Enter key) to add space

    • This breaks layout across devices or template changes.

  • Manually numbering lists

    • Word automatically handles numbering, even when lists change.

  • Changing font size/color to "fake" a heading

    • Use styles instead (Heading 1, Heading 2, etc.)

  • Centering too much text

    • Center only major titles (rarely body paragraphs)

Accessibility Foundations

Formatting affects accessibility just as much as it affects appearance.

Key principles:

  • Do not use color alone to communicate meaning.
  • Use proper list tools so screen readers read items in order.
  • Keep body text left-aligned for readability.
  • Maintain good color contrast. Your text should be easily visible against the background.
  • Avoid overusing bold or italic for long passages.

Quick Reference

Text Formatting

  • Bold, italicize, underline
  • Change font size and color
  • Use highlighting sparingly

Paragraph Formatting

  • Alignment: left, center, right, justify
  • Spacing: set line spacing to 1.15
  • Indentation: first-line or hanging indents

Lists

  • Create bulleted, numbered, and multilevel lists
  • Practice nesting list items
