Lesson 1: Navigating Word & Using Styles Effectively

This lesson introduces core navigation tools and the use of built-in Word styles.

1. Navigation Pane

The Navigation Pane helps you move quickly through your document elements, which is especially crucial when working with long documents.

  • Go to View > Navigation Pane
  • Use the Headings, Pages, and Results tabs to navigate

Try it!

  • Open your W-Lab1 starter document.
  • Use the Navigation Pane to jump to different sections.
  • Click each heading to jump through the document.

2. Using Find & Replace

The Find and Replace feature helps you quickly locate and modify text throughout your document.

  • Go to Home > Find & Replace
  • Use the Find option to locate specific text.
  • Use the Replace option to substitute text with another value.

Try it!

  • Open your W-Lab1 starter document.
  • Use the Find and Replace feature to locate and modify specific text.

Go-To Command

The Go-To command (shortcut: CTRL+G) allows you to quickly navigate to specific locations in your document.

  • Go to Home > Go To
  • Select a location type (e.g., Page, Section, Heading)

Try it!

  • Open your W-Lab1 starter document.
  • Use the Go-To command to jump to specific pages or sections.

3. Using Styles

Styles allow consistent formatting throughout a document.

  • Apply Heading 1, Heading 2, and Normal
  • Use styles to create structure and prepare for a Table of Contents
