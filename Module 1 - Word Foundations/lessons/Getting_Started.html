Pre-Lesson 1: Getting Started with Microsoft Word

This lesson introduces the basic features and functionality of Microsoft Word.

Start a New Document

To start a new document, click on File > New and select a blank document or a template.

Try it!

You can also use the keyboard shortcut Ctrl+N to quickly create a new document.

Open an Existing Document

To open an existing document, click on File > Open and select the document you want to open.

Try it!

You can also use the keyboard shortcut Ctrl+O to quickly open a document.

Save an Existing Document

To save an existing document, click on File > Save to save the document's changes.

Try it!

You can also use the keyboard shortcut Ctrl+S to quickly save a document.
