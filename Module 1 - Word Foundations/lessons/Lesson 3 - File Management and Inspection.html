Lesson 3: Saving, Sharing, Inspecting & Accessibility

This lesson prepares students for professional document management.

1. File Saving & Versioning

  • Save documents with clear naming conventions
  • Create versioned files (v1, v2, Final)

2. Exporting as PDF

Use File > Save As > PDF to export your final work.

3. Sharing Documents

Create shareable links with the correct permission settings.

4. Inspecting Documents

  • Run Document Inspector to remove metadata
  • Run Accessibility Checker to ensure inclusivity

5. Practice

Save a versioned document, export to PDF, and run both inspection tools.
