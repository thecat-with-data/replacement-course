# Module 3: Word Graphics & Collaboration

## Overview
Module 3 teaches visual document enhancement and collaborative review workflows in Microsoft Word. It covers graphics, layout, comments, track changes, document review, and accessibility checks.

## Learning outcomes
By the end of Module 3, students should be able to:
- Insert shapes, SmartArt, and images
- Wrap and position images
- Format text boxes
- Add alt text for accessibility
- Add comments and use track changes
- Accept and reject changes
- Compare and merge documents
- Re-check accessibility after edits
