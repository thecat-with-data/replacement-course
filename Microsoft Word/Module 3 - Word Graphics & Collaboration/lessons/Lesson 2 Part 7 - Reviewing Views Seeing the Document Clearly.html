Lesson 2 Part 7 - Reviewing Views: Seeing the Document Clearly

Word offers multiple review views.

Common Review Views

  • Simple Markup - clean view with revision indicators
  • All Markup - shows all changes explicitly
  • No Markup - preview of final document
  • Original - document before changes

Important Skill

You must know how to:

  • Toggle views while editing
  • Preview the final version before submission
