Lesson 1: Visual Elements & Layout Enhancement

By the end of this lesson, you will be able to:

  • Insert visual elements such as images, shapes, text boxes, and SmartArt
  • Position and format visual elements intentionally
  • Use visuals to improve clarity, not distract from content
  • Apply text wrapping and alignment correctly
  • Recognize and avoid common graphic-design mistakes in Word documents
