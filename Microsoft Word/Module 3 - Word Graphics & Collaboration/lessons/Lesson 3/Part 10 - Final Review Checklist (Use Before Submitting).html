Part 10 - Final Review Checklist (Use Before Submitting)

Before submitting W-Lab 5 or the Word Project , confirm:

  • All comments resolved or removed
  • Track Changes turned OFF (unless otherwise instructed)
  • Review views checked
  • Accessibility Checker run
  • Document Inspector run
  • Formatting consistent throughout

  • Required formats submitted (DOCX, PDF if applicable)
