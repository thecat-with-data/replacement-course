W-Lab 5: Graphics, Comments & Track Changes

Purpose

This lab develops your ability to enhance documents visually and collaborate professionally using Microsoft Word. You will practice inserting and formatting visual elements, adding comments for feedback, and managing document revisions using Track Changes. These skills reflect how Word is used in academic, professional, and workplace environments.

Learning Outcomes

Upon completion of this lab, you will demonstrate the ability to:

  • Insert illustrations and text boxes (A13)
  • Format illustrations and text boxes (A14)
  • Add text to graphic elements (A15)
  • Modify graphic elements purposefully (A16)
  • Add and manage comments (A17)
  • Manage change tracking accurately (A18)
  • Review documents for professionalism and readiness (A4 - reinforcement)

Getting Started

  1. Download the W-Lab 5 Starter Document.
  2. Save it immediately as:
    • Lastname_Firstname_W-Lab5.docx
  3. Complete all required tasks below using Microsoft Word.

Required Tasks

Complete all sections carefully. Each task is graded.

Visual Elements

Insert all of the following visual elements into the document at appropriate locations:

  1. One image
    • Photo, screenshot, or royalty-free image
    • Must support the surrounding text
  2. One shape
    • Rectangle, callout, arrow, or similar
    • Must contain text related to the document content
  3. One text box
    • Used for a summary, definition, or important note
  4. One SmartArt graphic
    • Used to represent a process, hierarchy, cycle, or relationship

Formatting & Layout

Format all inserted visual elements professionally:

  • Resize images proportionally (no stretching)
  • Apply appropriate picture or shape styles
  • Align visuals with margins or nearby text
  • Set text wrapping intentionally:
    • In Line with Text OR
    • Square / Top and Bottom (if appropriate)
  • Ensure visuals improve readability rather than clutter the page

Accessibility

For each visual element:

  • Add Alt Text
  • Describe the purpose of the visual, not just its appearance

Accessibility is required and graded.

Comments (Collaboration)

  1. Insert at least two comments in the document:
    • One comment suggesting an improvement
    • One comment asking a question or requesting clarification
  2. Comments must be:
    • Clear
    • Professional
    • Related to document content

Track Changes (Revision Simulation)

  1. Turn Track Changes ON
  2. Make all of the following types of edits:
    • One insertion
    • One deletion
    • One formatting change
  3. Ensure tracked changes are visible when submitted.

Review & Cleanup

Before submission:

  • Switch between:
    • All Markup
    • No Markup
  • Confirm that changes and comments display correctly
  • Do NOT accept or reject all changes unless instructed

Leave Tracked Changes ON for grading.

Deliverable

Upload one document:

  • Lastname_Firstname_W-Lab5.docx
  • Tracked changes and comments must remain visible

Assessment Checklist

Visual Elements

  • Image inserted appropriately (A13)
  • Shape with text included (A15)
  • Text box included (A15)
  • SmartArt used appropriately (A16)

Formatting & Layout

  • Visuals are aligned and sized correctly (A14)
  • Text wrapping applied intentionally (A14)
  • Consistent, professional visual style (A16)

Accessibility

  • Alt text added to all visual elements (A14/A16)

Collaboration Tools

  • Two meaningful comments added (A17)
  • Track Changes enabled and used correctly (A18)
  • At least three different types of revisions included (A18)

Professionalism

  • Naming convention followed
  • No unresolved technical issues
  • Document appears ready for pear/instructor review

Common Issues to Avoid

  • Using decorative images with no purpose
  • Stretching images out of proportion
  • Floating objects that disrupt text flow
  • Leaving track changes off
  • Using comments as permanent text
  • Forgetting alt text

Reflection

Ask yourself before submitting:

  • Do my visuals add value to the content?
  • Would this document be clear to someone else?
  • Does it look professional and intentional?
