Word Project

Purpose

This project demonstrates your full ability to create a professionally formatted, accessible, multi-section Word document using advanced features. It assesses your mastery of all Word outcomes (A1-A18). You will produce a 6-10 page professional report with multiple sections, heading structure, tables, figures, citations, a Table of Contents, a Table of Figures, graphics, comments, tracked changes, and accessibility-ready formatting.

Scenario

You have been hired as a Technical Documentation Specialist for a small organization. Your task is to create a professional, polished report that your supervisor can present to leadership.

Choose ONE topic (or get approval for your own):

  • Technology in Education
  • Cybersecurity Awareness for Employees
  • Data-Driven Decision Making
  • Artificial Intelligence in Business
  • Improving Workplace Communication
  • Digital Wellness & Productivity
  • OR a topic assigned by your instructor

Your report will include:

  • A title page
  • Multiple sections
  • References/citations
  • Graphics and diagrams
  • A clean, consistent format
  • Professional tone and structure

Required Components (Step-by-Step Tasks)

Below is your exact task list. Follow it carefully.

Document Setup (A1-A3)

  • Create a new document or use the provided Project Starter File.
  • Set:
    • Margins to Normal (1")
    • Line spacing to 1.15
    • Font: Calibri, 11 pt (unless specified otherwise)
  • Add:
    • A title page (no page number)
    • Your name
    • Course name
    • Submission date

Styles, Headings & Navigation (A1, A2, A6)

  • Build a logical heading structure using:
    • Heading 1 (main sections)
    • Heading 2 (subsections)
    • Heading 3 (optional)
  • Ensure the Navigation Pane cleanly reflects your structure.
  • Apply consistent Normal body text formatting.

Section Breaks & Page Numbering (A7)

Insert two different sections using Section Break (Next Page):

  • Section 1 > Title Page (no page number)
  • Section 2 > Main Content (page numbers start at 1)

Page numbers should be:

  • Bottom of page
  • Centered or right-aligned
  • Use standard Arabic numerals

Tables & Lists (A8, A9, A10)

Include ALL of the following:

  • One table with:
    • At least 3 columns x 3 rows
    • A Table Style (Header Row + Banded Rows recommended)
    • A meaningful caption (e.g. Table 1. Survey Responses)
  • One bulleted list
  • One numbered list
  • One multilevel list

References & Reference Tables (A11, A12)

Use Word's built-in tools to create:

  • At least three citations (APA or MLA recommended)
  • A correct Bibiliography / Works Cited
  • One footnote and one endnote
  • Insert:
    • Table of Contents
    • Table of Figures
  • Update all fields right before submitting.

Graphics, Illustration Formatting, Text Boxes (A13-A16)

Insert ALL of the following:

  • One picture (your own or royalty-free)
  • One shape (e.g. callout, rectangle)
  • One text box containing a key quote or summary
  • One SmartArt diagram

Format these objects by applying:

  • Text wrapping (e.g. Square, Tight)
  • Picture Style (drop shadow, border, etc.)
  • Shape fill & outline
  • SmartArt color & layout variations

Add alt text to every graphic object.

Comments & Track Changes (A17, A18)

Simulate a revision cycle:

You must include both versions:

  • Draft Version (with track changes visible)
  • Final Version (clean, all changes accepted)

In your draft:

  • Insert at least 3 comments
  • Make changes with Track Changes ON, including:
    • One insertion
    • One deletion
    • One formatting improvement

Save, Export, Inspect (A3, A4)

Before submitting:

  • Run Document Inspector
    • Remove personal information
  • Run Accessibility Checker
    • Fix at least one issue
  • Export your final version as a PDF.

Deliverables

Submit the following FOUR files:

  1. Lastname_Firstname_WordProject_Draft.docx
    • Contains tracked changes + comments
  2. Lastname_Firstname_WordProject_Final.docx
    • Clean, fully formatted document
  3. Lastname_Firstname_WordProject.pdf
    • Exported final PDF
  4. Lastname_Firstname_Reflection.txt
    • 150-200 word reflection answering:
      • What was the most challenging part?
      • What Word skills improved the most?
      • What would you do differently in a real workplace assignment?

Assessment Checklist

Document Structure and Navigation

  • Heading hierarchy applied correctly (A1, A2)
  • Navigation Pane reflects sections (A1, A2)
  • Section breaks & page numbering correct (A7)

Formatting & Content

  • Tables created + formatted (A8, A9)
  • Lists properly formatted (A10)
  • Graphics inserted + formatted (A13-A16)
  • Alt text added to all objects (A14-A16)

References & Metadata

  • Citations + bibliography (A11)
  • TOC + Table of Figures generated (A12)
  • Footnote & endnote included (A11)

Collaboration Tools

  • Comments added (A17)
  • Track Changes used accurately (A18)
  • Both draft + final versions submitted

Professionalism & Accessibility

  • Accessibility Checker results addressed (A4)
  • Inspector run & metadata cleaned (A4)
  • PDF included (A3)
