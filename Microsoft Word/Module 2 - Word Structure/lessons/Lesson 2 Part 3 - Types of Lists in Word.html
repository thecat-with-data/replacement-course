Lesson 2 Part 3 - Types of Lists in Word

Bulleted Lists

Use when:

  • Order does not matter
  • Items are equal in importance

Examples:

  • Features
  • Characteristics
  • Options

Professional tip : keep bullet text parallel in structure (.e.g. all start with verbs or nouns).

Numbered Lists

Use when:

  • Sequence matters
  • Instructions must be followed in order

Examples:

  1. Procedures
  2. Steps
  3. Processes

Word automatically updates numbering if you add, remove, or reorder steps. Manually typed numbers do not.

Multilevel Lists

Use when:

  • Items belong to a category
  • Steps contain sub-steps
  • An outline structure is required

Examples:

  • Project Phases
    • Planning
      • Research
      • Budgeting
    • Execution

Multilevel lists are essential for:

  • Outlines
  • Policies
  • Procedures
  • Technical documentation
