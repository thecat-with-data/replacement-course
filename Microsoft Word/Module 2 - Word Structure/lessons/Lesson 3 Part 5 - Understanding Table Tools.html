Lesson 3 Part 5 - Understanding Table Tools

When a table is selected, two tool sets appear:

Table Design

Controls appearance>:

  • Table Styles
  • Header row formatting
  • Banded rows or columns
  • Borders and shading

Table Layout

Controls structure:

  • Insert/delete rows or columns
  • Cell size (height/width)
  • Merge and split cells
  • Text alignment within cells

Design = how the table looks

Layout = how the table works
