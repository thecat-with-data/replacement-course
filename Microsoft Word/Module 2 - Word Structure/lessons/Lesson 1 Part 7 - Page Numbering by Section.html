Lesson 1 Part 7 - Page Numbering by Section

A very common requirement:

  • Title page: no page number
  • Body pages: start at page 1

How to Do This

  1. Insert a Next Page section break after the title page.
  2. Open the footer in the second section
  3. Turn off Link to Previous
  4. Insert page numbers
  5. Format page numbers > Start at 1

This workflow will appear in W-Lab 3 and the Word Project
