Part 3 - Ways to Create Tables in Word

Method 1: Insert > Table

  • Choose the number of rows and columns visually
  • Best for simple, predictable tables

Method 2: Insert > Table > Insert Table

  • Manually specify dimensions
  • Allows for greater precision

Method 3: Convert Text to Table

  • Converts existing text into a table
  • Requires consistent delimiters (tabs or commas)

This method is useful for cleaning up poorly formatted documents.
