Part 7 - Modifying Tables

You should also be comfortable adjusting tables after creation.

Common Modifications

  • Insert rows above/below
  • Insert columns left/right
  • Resize columns or rows
  • Merge header cells for clarity
  • Split cells to separate data

Avoid

  • Dragging table edges excessively
  • Stretching tables to fit the page instead of adjusting content
  • Using manual spacing inside cells
