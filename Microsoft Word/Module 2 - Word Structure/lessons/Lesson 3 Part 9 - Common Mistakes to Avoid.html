Lesson 3 Part 9 - Common Mistakes to Avoid

  • Using tables to align text
  • Skipping header rows
  • Mixing fonts and spacing
  • Applying too many styles
  • Making tables wider than the page
  • Using screenshots instead of actual tables

Accessibility Considerations

Accessible tables:

  • Include header rows
  • Present logical reading order
  • Avoid merged cells when possible
  • Use simple, consistent structures
