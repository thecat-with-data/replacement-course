Lesson 3 Part 2 - When to Use a Table (and When Not To)

Use a Table When:

  • Data needs to be compared
  • Information follows a row/column pattern
  • Multiple categories share attributes

Do NOT Use a Table When:

  • Aligning text visually (use tabs or alignment instead)
  • Creating page layout designs
  • For short lists or paragraphs

Rule of Thumb:

If content could be read linearly as a paragraph, a table is probably not needed.
