Lesson 1 Part 3 - Types of Section Breaks

Word provides several types of section breaks. In this course, we focus on two.

Next Page Section Break

  • Starts a new section on a new page
  • Most common and most useful

Use it when:

  • Creating a title page
  • Starting main content
  • Changing page numbering

Continuous Section Break

  • Starts a new section on the same page
  • Used for subtle layout changes

Use it when:

  • Changing number of columns
  • Altering headers mid-page
  • Adjusting formatting without adding a page
