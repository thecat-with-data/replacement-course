Lesson 1: Document Sections & Layout Control

This lesson introduces the concept of document sections and how to control the layout of your Word documents.

By the end of this lesson, you should be able to:

  • Explain the purpose of sections in Word documents
  • Distinguish between page breaks and section breaks
  • Insert and manage different types of section breaks
  • Control headers, footers, and page numbering by section
  • Apply professional layout conventions used in long documents

Why Document Sections Matter

In Word, sections - not pages - control layout.

Many students assume that each page behaves independently. This is not true. Word treats a document as a continuous flow of content unless you explicitly create section boundaries.

Sections allow you to:

  • Restart or suppress page numbers
  • Use different headers and footers
  • Apply different layouts in the same document
  • Separate front matter from main content

These skills are essential for:

  • Academic writing projects
  • Business reports
  • Manuals and proposals
  • The Word Project later in this course

Pages vs. Sections: A Critical Distinction

Page Break

  • Starts content on a new page
  • Does not allow layout changes
  • Best for:
    • Starting a new chapter on a new page
    • Keeping related content together

Section Break

  • Divides a document into layout zones
  • Restarting page numbers
  • Title pages vs body content

Key Takeaway

If you want to change how something looks, you need a section break, not a page break.

Types of Section Breaks

Word provides several types of section breaks. In this course, we focus on two.

Next Page Section Break

  • Starts a new section on a new page
  • Most common and most useful

Use it when:

  • Creating a title page
  • Starting main content
  • Changing page numbering

Continuous Section Break

  • Starts a new section on the same page
  • Used for subtle layout changes

Use it when:

  • Changing number of columns
  • Altering headers mid-page
  • Adjusting formatting without adding a page

Mini-Demo: Inserting a Section Break (Next Page)

  1. Click where you want the new section to begin
  2. Go to Layout > Breaks
  3. Under Section Breaks, select Next Page
  4. Confirm the document jumps to a new page

This creates a new section with its own layout controls.

Headers, Footers & Section Linking

Each section can have its own:

  • Header
  • Footer
  • Page numbers

By default, sections are linked to each other.

"Link to Previous"

  • When ON, formatting carries over.
  • When OFF, the section becomes independent.

To change headers/footers in one section without affecting others, turn off Link to Previous.

Mini-Demo: Unlinking Headers

  1. Double-click the header area
  2. Select Link to Previous (turn it off)
  3. Edit the header only for that section

This is how title pages remain unnumbered while body pages are numbered.

Page Numbering by Section

A very common requirement:

  • Title page: no page number
  • Body pages: start at page 1

How to Do This

  1. Insert a Next Page section break after the title page.
  2. Open the footer in the second section
  3. Turn off Link to Previous
  4. Insert page numbers
  5. Format page numbers > Start at 1

This workflow will appear in W-Lab 3 and the Word Project

Common Mistakes

  • Using page breaks instead of section breaks
  • Forgetting to turn off Link to Previous
  • Trying to remove page numbers by manually deleting them
  • Adding multiple unnecessary section breaks
  • Believing each page has its own layout unit

Recognizing these mistakes early will save hours of frustration.

Accessibiity & Professional Standards

Sections help with accessibility by:

  • Supporting logical reading order
  • Ensuring headings and page numbers follow structure
  • Improving navigation for screen readers

Structured documents are easier to:

  • Read
  • Review
  • Grade
  • Share

  3. Start main content on the next page
  4. Turn off Link to Previous
  5. Add page numbering starting at 1

Lesson Summary

In this lesson, you learned that:

  • Sections are the foundation of document layout

  • Headers, footers, and page numbers depend on sections
