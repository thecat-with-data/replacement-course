Module 2: Word Structure

Module Learning Outcomes

By the end of this module, students will be able to:

  • Create and manage document sections
  • Control headers, footers, and page numbering
  • Build and format tables appropriately
  • Use lists to show hierarchy and structure
  • Insert and manage citations and references
  • Generate and update a Table of Contents and other reference tables
