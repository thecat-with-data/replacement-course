Lab 3: Sections, Lists & Tables

Purpose

This lab develops your ability to structure professional Word documents using section breaks, lists, and tables. These skills are critical for long-form academic and workplace documents.

Outcomes

  • A7 Create and configure document sections
  • A8 Create tables
  • A9 Modify tables
  • A10 Create and modify lists

Instructions

  1. Download the W-Lab 3 starter document.
  2. Complete all tasks listed in the checklist using Microsoft Word.
  3. Save your document using the required naming convention.
  4. Upload your completed file to this assignment.

Required Tasks

  • Insert multiple section breaks and configure headers/footers.
  • Create bulleted, numbered, and multilevel lists.
  • Insert and format at least two tables.
  • Modify table structure (rows, columns, merged cells).
