Lab 2: Save, Share & Inspect

Outcomes

A3, A4

Purpose

This lab teaches file management: saving versions, exporting, sharing, and inspecting document metadata.

Tasks

  1. Open the provided W-Lab2-Module1.docx starter document.
  2. Save the file locally with correct naming: Lastname_Firstname_W-Lab2.docx
  3. Save a second version named Lastname_Firstname_W-Lab2_v2.docx
  4. Export the document as a PDF.
  5. Create a Sharable Link (OneDrive or equivalent):
    • Set permissions to "Can View"
    • Copy the link into the last page of the document.
  6. Run Document Inspector:
    • Remove personal information
    • Remove comments/hidden data if detected
  7. Run Accessibility Checker and fix at least one issue.

Deliverables

  • Upload:
    • Lastname_Firstname_W-Lab2.docx
    • Lastname_Firstname_W-Lab2.pdf
  • Share link pasted into the document.

Checklist

  • Correct naming conventions (A3)
  • PDF exported (A3)
  • Share link added (A3)
  • Document Inspector run & issues resolved (A4)
  • Accessibility Checker reviewed (A4)
