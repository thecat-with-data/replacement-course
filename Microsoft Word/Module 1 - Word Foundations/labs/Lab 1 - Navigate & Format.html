Lab 1: Navigate & Format

Outcomes

A1, A2, A5, A6

Purpose:

This lab builds foundational Word skills: navigating documents, applying formatting, and working with text and paragraphs.

Tasks

  1. Open the provided W-Lab1-Module1.docx starter document.
    • Select 'View' from the Ribbon, then select 'Navigation Pane' in the 'Show' section.
  2. Use the Navigation Pane to move to each heading.
  3. Apply the following to the document text:
    • Change paragraph alignment (left, center, right)
    • Update line spacing to 1.15
    • Apply First Line Indent to one paragraph
    • Use Find and Replace to change a repeated word to something new
    • Use Format Painter, then apply the formatting to another paragraph.
  4. Insert two new paragraphs under the existing placeholder text.
  5. Apply:
    • A built-in Style (Heading 1, Heading 2, Normal)
    • Bold, italics, underline
    • Bulleted list (one item must be nested)
  6. Insert a page break at an appropriate point.
  7. Review your formatting for consistency.

Deliverables

Upload Lastname_Firstname_Lab1.docx

Checklist

  • Navigation Pane used correctly (A1)
  • Text & paragraph formatting applied correctly (A2, A6)
  • New paragraphs inserted correctly (A5)
  • Lists properly formatted (A6)
  • Page break correctly placed
  • Style hierarchy applied correctly (A2)
