# Module 1: Word Foundations

## Overview
This module covers the fundamentals of Microsoft Word. It introduces document navigation, formatting, insertion of basic content, file management, sharing, and document inspection.

## Learning outcomes
By the end of Module 1, students should be able to:
- Navigate a Word document using built-in tools
- Format text and paragraphs using standard conventions
- Insert text, lists, and page breaks
- Save, version, export, and share Word documents
- Use Document Inspector and Accessibility Checker to prepare documents for submission
