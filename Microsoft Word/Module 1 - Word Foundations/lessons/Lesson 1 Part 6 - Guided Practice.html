Lesson 1 Part 6 - Guided Practice

Try adding headings and observing how they appear in the Navigation Pane.

  1. Add three headings using Heading 1.
  2. Add two subheadings under the second heading using Heading 2.
  3. Add a short paragraph under each subheading.
  4. Turn on Navigation Pane and reorder headings by dragging.
  5. Practice making:
    • One bulleted list
    • One numbered list
    • One multilevel list
  6. Format one paragraph with:
    • Line spacing 1.15
    • First line indent
    • 6pt space after paragraph
