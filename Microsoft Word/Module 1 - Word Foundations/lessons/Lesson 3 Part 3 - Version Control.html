Lesson 3 Part 3 - Version Control

Versioning allows you to track changes and recover from mistakes.

When to Create a New Version

  • Before major edits
  • Before revisions based on feedback
  • Before submitting a final draft

Simple Versioning System

  • _v1
  • _v2
  • _Draft
  • _Final
Example:Lastname_Firstname_W-Lab2_v2.docx
