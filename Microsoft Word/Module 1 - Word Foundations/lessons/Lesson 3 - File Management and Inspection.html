Lesson 3: File Management and Inspection

This lesson prepares students for professional document management.

By the end of this lesson, students will be able to:

  • Save Word documents using clear naming conventions
  • Create and manage document versions
  • Export documents to PDF
  • Share documents safely using view/edit permissions
  • Inspect Word documents for metadata, privacy, and accessibility issues
  • Explain why inspection is required in academic and professional settings
