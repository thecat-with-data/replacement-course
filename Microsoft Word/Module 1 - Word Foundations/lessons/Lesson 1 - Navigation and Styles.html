Lesson 1: Navigation and Styles

This lesson introduces core navigation tools and the use of built-in Word styles, as well as going into the 'why' behind their use.

By the end of this lesson, you will be able to:

  • Navigate a Word document using built-in tools.
  • Format text and paragraphs using standard conventions.
  • Insert new text, lists, and page breaks.
