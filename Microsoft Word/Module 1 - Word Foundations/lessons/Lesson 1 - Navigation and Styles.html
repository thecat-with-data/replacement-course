Lesson 1: Navigating Word & Using Styles Effectively

This lesson introduces core navigation tools and the use of built-in Word styles, as well as going into the 'why' behind their use.

By the end of this lesson, you will be able to:

  • Navigate a Word document using built-in tools.
  • Format text and paragraphs using standard conventions.
  • Insert new text, lists, and page breaks.

Navigation Pane

The Navigation Pane helps you move quickly through your document.

Why Does It Matter?

Navigation and styles matter in the real world because they improve document organization and accessibility.

Who Uses Them?

Navigation and styles are used by professionals who need to manage and maintain large documents, ensuring consistency and ease of access.

Examples of documents which rely on structure include:

  • Long reports
  • Manuals
  • Legal documents
  • Academic papers
  • Business proposals

The 'Heading' Style

Using the heading style has several benefits, including:

  • Instant navigation through the document
  • Automatic generation of a Table of Contents
  • Consistent formatting across the document
  • Cleaner accessibility for screen readers

Styling Best Practices

  • Never manually bold, size, and center your titles. Use Heading 1.
  • Never press Enter to advance a page. Use page breaks instead.
  • Don't space paragraphs with Enter. Use paragraph spacing.
  • Don't manually indent lists. Use the list tools.
  • Use the Styles Pane instead of hard formatting whenever possible.

Styling Common Mistakes

  • Manually spacing paragraphs with blank lines.
  • Using spaces or tabs to align text.
  • Highlighting entire paragraphs and applying multiple manual formats.
  • Forgetting to update Styles when making changes.
  • Typing a heading and bolding it instead of using Heading 1.

Accessibility Considerations

  • Headings help screen readers navigate the document.
  • Avoid using color alone to convey meaning.
  • Ensure sufficient contrast between text and background.
  • Images must have descriptive alt text (if included)
  • List tools ensure proper reading order.

Using Styles

Styles allow consistent formatting throughout a document.
