Lesson 2 Part 5 - Common Mistakes Beginners Make

  • Using spaces or tabs to align text

    • Use the formatting tools instead.

  • Using multiple Returns (Enter key) to add space

    • This breaks layout across devices or template changes.

  • Manually numbering lists

    • Word automatically handles numbering, even when lists change.

  • Changing font size/color to "fake" a heading

    • Use styles instead (Heading 1, Heading 2, etc.)

  • Centering too much text

    • Center only major titles (rarely body paragraphs)
