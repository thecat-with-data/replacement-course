Lesson 3 Part 2 - File Naming Best Practices

Unless otherwise instructed, you will use the following naming convention:

Lastname_Firstname_AssignmentName.docx

What a Good File Name Includes

    Who created it
    What it is
    Which version it is

Avoid These

  • final.docx
  • assignment.docx
  • lab2newnewFINAL.docx

These create confusion and errors, especially during grading or collaboration.
